Changelog

Version 1.0

  • - Plant EverythingThis feature allows you to plant all types of crops in your farm.
  • - Auto FeedThis feature automatically feeds your animals so you don't have to manually do it.
  • - Multi CraftThis feature allows you to craft multiple items at once.
  • - Fascinating Carry WeightThis feature increases your carrying capacity.
  • - Mass FarmingThis feature allows you to farm a large amount of crops at once.
  • - Automatic FuelThis feature automatically refuels your items so you don't have to manually do it.
  • - Toggle MovementThis feature allows you to toggle movement for certain activities.
  • - Multi User ChestThis feature allows multiple users to access a single chest.
  • - Craft from ContainersThis feature allows you to craft items directly from containers.
  • - Odin BannersThis feature allows you to decorate with Odin-themed banners.
  • - Viking SailsThis feature allows you to decorate with Viking-themed sails.
  • - Comfy SignsThis feature allows you to place signs around your base.
  • - Simple Auto SortThis feature allows you to automatically sort your items.
  • - Quick Stack Store Trash RestockThis feature allows you to quickly store, stack, trash, and restock your items.
  • - Adventure BackpacksThis feature allows you to carry more items with specialized backpacks.
  • - Odin CampsiteThis feature allows you to place an Odin-themed campsite.
  • - RecycleThis feature allows you to recycle items.
  • - Use Equipment In WaterThis feature allows you to use equipment while in water.
  • - Floating ItemsThis feature allows you to collect floating items.
